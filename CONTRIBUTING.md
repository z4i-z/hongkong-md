## 為何 HongKong.md

香港是世界上最多元、最矛盾、最容易被誤解的城市之一。

七百五十萬人在 1,106 平方公里的土地上生活，在地產霸權與金融中心的夾縫中創造了獨一無二的文化身份。飲茶、港產片、廣東歌、公屋生活——這些唔係旅遊景點，係香港人日常的一部分。

HongKong.md 是一個開源的、AI 友好的香港知識庫，用策展的眼光而非旅遊指南的方式，呈現這座城市最真實的面貌。

## 貢獻方式

### 最簡單：告訴我們你知道的事

任何人都可以透過填寫[表單](https://hongkong.md/contribute)提供資料。我們的團隊會整理並寫成文章。

### 翻譯

HongKong.md 的原文是繁體中文（SSOT），我哋需要：
- 英文翻譯
- 日文翻譯
- 其他語言

請先閱讀 [TRANSLATE_PROMPT.md](./docs/prompts/TRANSLATE_PROMPT.md)。

### 直接提交 Pull Request

適合 developer：

1. Fork 此 repository
2. 在 `knowledge/` 目錄下建立或修改 `.md` 文件
3. 提交 Pull Request

## 資料來源標準

所有內容都需要可靠來源。我哋接受：

- 新聞機構報道（附連結）
- 學術論文
- 政府統計數據
- 親身經歷（注明「作者親歷」）

不接受：

- 匿名消息人士（除非是重大事件且有其他資料佐證）
- 未經證實的傳聞
- 純粹個人意見
