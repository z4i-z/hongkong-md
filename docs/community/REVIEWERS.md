# PR 審核指南 — HongKong.md

> 每一個 PR 都是一次策展機會。審核者的任務是確保內容品質，唔係挑剔。

---

## 審核原則

1. **有錯就要改** — 事實錯誤、拼寫錯誤、格式問題必須修正
2. **風格問題建議改** — 寫作風格、語氣、節奏可以提出建議，但不是命令
3. **創作自由要尊重** — 對方對某個主題的切入角度不同？可以討論，但唔好强迫
4. **有溫度** — HongKong.md 的文章要有溫度，冷冰冰的百科式文字唔合格

---

## 審核清單

### 必須通過（否則要求修改）

- [ ] 有可靠的來源連結嗎？（至少 3 個）
- [ ] Frontmatter 格式正確嗎？（title、description、tags、category）
- [ ] 沒有抄襲嗎？（可以用簡單的句子搜索驗證）
- [ ] 沒有惡意攻擊或誹謗內容嗎？
- [ ] 語言是繁體中文嗎？（唔接受簡體字）

### 建議通過（可以合併，但建議改進）

- [ ] 有具體的數字、年份、人名嗎？
- [ ] 有策展觀點（📝 標記）嗎？
- [ ] 段落長度適中嗎？（平均 80-150 字）
- [ ] 開場有具體事實嗎？（唔好「香港是一個...」開頭）

---

## 常見拒絕原因

1. **沒有來源** — 「我認為...」不是來源
2. **抄襲** — 直接複製維基百科內容
3. **塑膠句** — 「隨著...」「近年來...」「眾所周知...」大量出現
4. **冇溫度** — 讀起來像機器寫的百科全書
5. **太短** — 少於 500 字又沒有独特觀點的文章

---

## 如何給予反饋

尽量用鼓勵的方式提出建議：

- ✅ 「呢個角度好有趣，但如果加多一個具體例子會更突出」
- ❌ 「呢個寫得唔好，要改」
