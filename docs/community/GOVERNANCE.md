# 治理方針 — HongKong.md

> HongKong.md 是一個開放的協作項目，所有決策都基於共識與透明度。

---

## 核心理念

HongKong.md 的目標是建立一個可信、深入、富有溫度的香港知識庫。我們相信：

1. **知識應當自由** — 內容採用 CC BY-SA 4.0，每個人都可以自由引用和改編
2. **多元聲音** — 香港的歷史和現實是複雜的，我們歡迎不同的觀點
3. **品質優先** — 數量不是目標，質量才是
4. **持續迭代** — 知識庫是一個生命體，會隨時間成長和演化

---

## 貢獻者角色

### 讀者（Reader）

任何閱讀 HongKong.md 的人。可以提出更正建議、回報錯誤、提供新的素材。

### 貢獻者（Contributor）

通過 PR 或 Issue 提交內容的人。所有人員一視同仁，沒有高低之分。

### 維護者（Maintainer）

負責審核 PR、維護文檔、推動項目發展的人。目前由 z4i-z（@z4i-z）維護。

---

## 決策機制

HongKong.md 的決策遵循以下原則：

1. **共識優先** — 重大決策需要社群討論達成共識
2. **文檔化** — 所有重要決策都記錄在文檔中
3. **透明** — 決策過程和理由公開可查

### 誰可以合併 PR？

目前只有維護者（z4i-z）有權限合併 PR。

### 如何提議新政策？

1. 開一個 GitHub Issue
2. 描述你的提案和理由
3. 等待社群討論
4. 維護者最終決定

---

## 內容政策

### 可以發布的內容

- 有可靠來源的歷史、地理、社會、文化資訊
- 親身經歷（注明「作者親歷」）
- 基於事實的分析和觀點

### 不可以發布的內容

- 沒有任何來源支持的主張
- 惡意攻擊或誹謗
- 抄襲的內容
