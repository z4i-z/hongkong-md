## Why HongKong.md?

香港的面積不過 1,106 平方公里，卻承載着全球最密集的天際線、最貴的房地產市場、以及七百五十萬種生活方式。

HongKong.md 是一個開源的、AI 友好的香港知識庫。不是旅遊指南，不是政治宣傳。是**策展的文學展覽**，呈現這座城市最值得被記住的面貌。

**預設使用繁體中文** — 這座城市的主體語言是粵語，書面語則使用繁體中文。

## 🤝 如何貢獻

### 最簡單：告訴我們你知道的事

填寫[表單](https://hongkong.md/contribute)，我哋會整理並寫成文章。

### AI 輔助

把以下文字貼比你喜歡的 AI（ChatGPT / Claude / Gemini），佢會引導你完成：

> 請閱讀 https://raw.githubusercontent.com/z4i-z/hongkong-md/main/docs/prompts/CONTRIBUTE_PROMPT.md ，然後引導我為 HongKong.md 撰寫一篇香港主題文章。

### 直接 Fork & PR

適合 developer：

1. Fork 此 repository
2. 在 `knowledge/` 目錄下建立或修改 `.md` 文件
3. 提交 Pull Request
